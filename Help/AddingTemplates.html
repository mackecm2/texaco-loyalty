To add a template

  • Create the document in Microsoft Word as a .DOT template file.
  • Fields are inserted as bookmarks (Insert->Bookmarks).
  • Go to Tools->Options->View and check bookmarks.
  • The available fields are:
  • bName
  • bSalute
  • bAddress1
  • bSystemdate
  • bAddress2
  • bCardno
  • bAddress3
  • bBalance
  • bAddress4
  • bBalDate
  • bAddress5
  • bLetterNo
  • bPostcode
  • Save the document on your hard disk.
  • Type a description (this will be displayed in the selection list).
  • Either type in a filename already on the server,
  • or, press the button to select the file you created.
  • Press the button.
