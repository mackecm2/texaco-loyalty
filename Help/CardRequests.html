To download a request batch

  • Card Requests are organised into batches. Clicking on "new batch" will create and download a new batch of requests. Clicking on any previous batch date will download that file.
  • Save these files on your local machine.
  • Load them into your card request database.

    After a batch has been processed and the response file has been returned then the batch will show how many of the original requests haven't been processed.
