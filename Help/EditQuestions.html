Manage questions

Create/Modify Question

  • Type in the question you would like the operator to ask.
  • Select how often you wish the question verified.
  • Select a question type:
    • Boolean (Yes/No).
    • Integer (Whole numbers only).
    • Text (Free form text).
    • Drop Down List.
    The bottom section will become available on selecting "drop down lists".
  • "Option" is the option that the operator will read out/select from.
  • "Response Value" will be the code entered into the database.
  • As you fill up rows, more rows should appear.

  • Press button when you have finished.
