Configuration for mail merge.

In order to run the mail merge process, it is necessary to:
  • have Microsoft Word installed on your machine,
  • have printer that you wish to print to as the default printer,
  • configure the browser to allow "active X controls". Then
    1. Goto Tools --> Internet options
    2. Select the Security Tag
    3. Select Trusted sites
    4. Press the sites button
    5. Uncheck Require server verification...
    6. Add 81.171.171.25.
    7. Press OK
    8. Select Custom Level
    9. Select Low in the "reset to:" box
    10. Press reset
    11. Press OK
