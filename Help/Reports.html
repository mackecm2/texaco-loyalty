To request a report.

  • Select the type of report (currently only Fraud exists).
  • Click the button.
  • The report will be queued to be run (Pending).
  • It's possible to delete a report while it's still in the pending state.

    To view a report

  • Once a report has been run (Success) you can click on it.
  • This will allow you to download the report as a csv file.
