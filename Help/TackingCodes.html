To Add a new code

  • Type in the description in the "New Description" box.
  • Press .
  • Duplicate entries will be rejected.

    To Change a code

  • Click on the item in the list.
  • Edit the description.
  • Press button.
  • This change will be reflected in all exsting records.

    To Remove a code

  • Uncheck the checkbox next to the item.
  • Press button.
  • Codes are only marked as deleted - all exsting records will still be present.
