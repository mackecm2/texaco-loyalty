To update a template

To update a template, you open it in Word, make your changes, then upload it back to the server. To do this process:
  • Select the template from the list.
  • Press the button.

    Microsoft Word should start up.
  • Go to Tools->Options->View
  • Check "bookmarks" checkbox.
  • Make the changes to the template.
  • Save the template on your hard disk.
  • Close Word.

    Then, back in Internet Explorer,
  • Press the button to select the file you saved
  • Press the button to complete the update of the template.
